# Admob Sample
Admob adaptive banner ad, rewarded interstitial ad and interstitial ad Android project


## Requirements
```bash
Android Studio Chipmunk 2021.2.1 or higher,
Gradle version: 7.2.0,
SDK API 32,
Platform Tools: 32.0.0,
AndroidX libraries,
Kotlin
```
